About GO-CREW

GO-CREW is a cutting-edge Retrieval-Augmented Generation (RAG) software implemented in Golang. It offers a seamless, offline solution for advanced text processing and generation, with optional online capabilities.

Features

  • Offline operation using local Ollama models
  • Local ChromaDb (encrypted/raw) for data storage
  • Process PDF and TXT files
  • URL content processing (when online)
  • Wikipedia data analysis (when online)
  • YouTube video transcription analysis (when online)
  • No Docker, Python, or Hugging Face dependencies
  • Standalone application - no browser required
